🎓 Flask學生管理系統

專案架構與開發流程完整指南

🏗️ 專案結構

核心配置

  • config.py - 數據庫連接、安全密鑰、分頁設置
  • manage.py - 數據庫初始化、創建管理員、Flask命令行
  • requirements.txt - Flask核心、SQLAlchemy ORM、Flask-Login、Flask-WTF

應用工廠

  • app/__init__.py - Flask實例化、擴展初始化、藍圖註冊、全局配置

📊 數據模型層

app/models/__init__.py

  • User - 認證功能、角色管理、密碼加密
  • Student - 學籍信息、個人資料、班級關聯
  • Teacher - 職員信息、部門關聯、班級管理
  • Class - 班級信息、師生關聯、部門歸屬
  • Department - 組織架構、層級管理

⚙️ 業務邏輯層

認證模組 (app/auth/)

  • routes.py - 登錄/登出、用戶註冊、會話管理
  • forms.py - 登錄表單、註冊表單、字段驗證

學生管理 (app/student/)

  • routes.py - 學生列表、新增學生、編輯學生、查看詳情
  • forms.py - 學生信息表單、搜索表單、驗證規則
  • decorators.py - 角色檢查、數據過濾、訪問控制

其他管理模組

  • app/teacher/ - 教師CRUD、表單管理、權限裝飾器
  • app/classes/ - 班級操作、班級表單、訪問控制
  • app/user_management/ - 用戶CRUD、用戶表單、管理權限

🎨 前端展示層

模板系統 (app/templates/)

  • base.html - 導航結構、樣式引入、JavaScript載入
  • auth/ - 登錄頁面、註冊頁面
  • student/ - 列表頁面、詳情頁面、表單頁面
  • teacher/、classes/、user_management/ - 各功能模組頁面

靜態資源 (app/static/)

  • CSS樣式、JavaScript腳本、圖片資源
