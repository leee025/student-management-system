🎓 Flask學生管理系統

專案架構與開發流程指南

📋 專案概述

這是一個基於Flask框架開發的完整學生管理系統，採用模組化設計和現代Web開發最佳實踐。系統支援多角色權限管理，提供學生、教師、班級和用戶的完整CRUD操作。

Flask 2.3.3 SQLAlchemy Flask-Login Flask-WTF MySQL Bootstrap

🏗️ 專案結構

核心配置文件

  • config.py - 應用配置（數據庫連接、安全密鑰、分頁設置）
  • manage.py - 管理腳本（數據庫初始化、創建管理員）
  • requirements.txt - 依賴管理
  • run.py - 應用啟動入口

應用工廠模式

  • app/__init__.py - 應用創建、擴展初始化、藍圖註冊

🚀 開發流程

環境準備

# 創建虛擬環境 python -m venv venv # 激活虛擬環境 (Windows) venv\Scripts\activate # 安裝依賴 pip install -r requirements.txt

數據庫配置

  • 配置MySQL連接設置
  • 設置環境變量
  • 修改config.py中的數據庫連接字符串

初始化階段

# 初始化數據庫 python manage.py init_db # 創建管理員賬戶 python manage.py create_admin # 測試運行 python run.py

開發階段

  • 模型設計 - 定義數據結構、建立關聯關係
  • 業務邏輯 - 路由設計、表單處理、權限控制
  • 前端開發 - 模板設計、樣式美化、交互功能

測試部署

  • 功能測試 - 單元測試、集成測試、用戶測試
  • 生產部署 - 環境配置、數據遷移、服務啟動
